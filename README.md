# Fitness Tracker App

A modern fitness tracking web application built using HTML, CSS and JavaScript.

## Features

- Activity Management (CRUD)
- Goal Tracking
- Analytics Dashboard
- BMI Calculator
- Water Intake Tracker
- Achievement System
- Dark/Light Theme
- Local Storage Persistence
- JSON Import/Export
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js

## How to Run

Open `index.html` in your browser.
